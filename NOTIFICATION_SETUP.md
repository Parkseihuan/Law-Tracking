# 간편한 알림 설정 가이드

이메일 설정이 복잡하다면 더 쉬운 방법들을 사용하세요!

## 추천 방법

### 1. Discord 웹훅 (★ 가장 쉬움!)

**장점:**
- ✅ 설정이 매우 간단 (1분 안에 완료)
- ✅ 무료
- ✅ 비밀번호나 앱 비밀번호 필요 없음
- ✅ PC/모바일 모두 알림 받기 가능

**설정 방법:**

1. Discord에서 서버 만들기 (또는 기존 서버 사용)
2. 채널 설정 > 연동 > 웹후크 만들기
3. 웹후크 URL 복사
4. 시스템 설정 페이지에서:
   - **Discord 알림 활성화** 체크
   - **웹훅 URL** 붙여넣기
   - **저장** 클릭

**웹훅 URL 예시:**
```
https://discord.com/api/webhooks/123456789012345678/abcdefghijklmnopqrstuvwxyz...
```

---

### 2. Telegram Bot (★ 쉬움)

**장점:**
- ✅ 모바일에서 즉시 알림
- ✅ 무료
- ✅ 한국에서 많이 사용

**설정 방법:**

1. Telegram에서 @BotFather 검색
2. `/newbot` 명령어로 새 봇 생성
3. 봇 이름 설정 (예: "내 법령 추적")
4. 봇 토큰 복사 (예: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. 만든 봇과 대화 시작 (/start)
6. Chat ID 확인:
   - https://api.telegram.org/bot**BOT_TOKEN**/getUpdates 접속
   - `"chat":{"id":123456789` 부분에서 숫자 복사
7. 시스템 설정 페이지에서:
   - **Telegram 알림 활성화** 체크
   - **Bot 토큰** 입력
   - **Chat ID** 입력
   - **저장** 클릭

---

### 3. Slack 웹훅

**장점:**
- ✅ 회사/팀에서 사용하기 좋음
- ✅ 웹훅 방식으로 간단

**설정 방법:**

1. Slack Workspace에서 Apps 검색
2. "Incoming Webhooks" 앱 설치
3. 채널 선택 후 웹훅 URL 생성
4. 시스템 설정 페이지에서:
   - **Slack 알림 활성화** 체크
   - **웹훅 URL** 입력
   - **저장** 클릭

## 자주 묻는 질문

**Q: 여러 개를 동시에 활성화할 수 있나요?**
A: 네! Discord, Telegram, 이메일을 모두 활성화하면 세 곳 모두로 알림이 발송됩니다.

**Q: 비용이 발생하나요?**
A: 아니요! Discord, Telegram, Slack 모두 무료입니다.

**Q: 모바일에서도 알림을 받을 수 있나요?**
A: 네! Discord 앱과 Telegram 앱을 설치하면 모바일에서도 푸시 알림을 받을 수 있습니다.

**Q: 어떤 방법이 가장 좋나요?**
A:
- **개인 사용**: Discord 또는 Telegram 추천
- **팀 사용**: Slack 추천
- **공식 기록용**: 이메일 추천

**Q: 설정 후 테스트는 어떻게 하나요?**
A: 설정 페이지에서 "수동 체크" 버튼을 클릭하면 즉시 알림이 발송됩니다.
