# 법령 일괄 추가 가이드

여러 법령을 한꺼번에 추가하고 관리하는 방법입니다.

---

## 🚀 방법 1: 미리 준비된 법령 목록 사용 (가장 쉬움) ⭐

교육 관련 주요 법령 10개가 미리 준비되어 있습니다.

### 실행 방법

```cmd
python bulk_add_laws.py
```

### 선택 옵션

실행하면 3가지 옵션이 나타납니다:

```
1) 모두 추가          - 10개 법령 전부 추가
2) 선택해서 추가      - 원하는 법령만 선택
3) 취소              - 종료
```

#### 옵션 1: 모두 추가

```
번호를 입력하세요 (1/2/3): 1
```

→ 10개 법령이 모두 추가됩니다.

#### 옵션 2: 선택해서 추가

```
번호를 입력하세요 (1/2/3): 2
선택: 1,3,5
```

→ 1번, 3번, 5번 법령만 추가됩니다.

**범위로 입력도 가능:**
```
선택: 1-5
```
→ 1번부터 5번까지 추가됩니다.

**조합도 가능:**
```
선택: 1-3,5,7-9
```
→ 1,2,3,5,7,8,9번 추가됩니다.

---

## 📝 방법 2: 법령 목록 파일 수정하여 사용 (커스터마이징)

원하는 법령을 직접 추가하려면:

### Step 1: law_list.txt 파일 열기

메모장이나 VSCode로 `law_list.txt` 파일을 엽니다.

```cmd
notepad law_list.txt
```

### Step 2: 법령 추가/삭제

```txt
# === 교육 기본 법령 ===
사립학교법
고등교육법
교육기본법

# === 내가 추가하고 싶은 법령 ===
평생교육법
학점인정 등에 관한 법률
```

- 각 줄에 법령명 하나씩 입력
- `#`로 시작하는 줄은 주석 (무시됨)
- 빈 줄도 무시됨

### Step 3: 저장 후 실행

```cmd
python add_laws_from_file.py
```

파일에 입력한 법령들이 모두 추가됩니다!


## ⚠️ 주의사항

### 1. 법령명은 정확하게 입력

❌ 잘못된 예:
```
사립학교법법
사립 학교법
```

✅ 올바른 예:
```
사립학교법
```

### 2. 한 번에 너무 많이 추가하지 마세요

- API 호출 제한이 있을 수 있음
- 추천: 한 번에 10-20개 정도

### 3. 이미 추가된 법령은 자동으로 건너뜀

```
⏭️  이미 추적 중 - 건너뛰기
```

중복 걱정 없이 안전하게 실행 가능!
